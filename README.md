Rest API for a simple book app 

- Spring MVC
- Hibernate 
- Mysql 

Database Setup

- Books has four fields {id,name,author,category}
- You can change your db properties in config.properties file

Rest APIs

- GET /api/books

- GET /api/books/{id}

- POST /api/books

- PUT /api/books

- DELETE /api/books/{id}
